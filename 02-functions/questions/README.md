# 📘 Function Practice Questions

This folder contains hands-on practice problems based on JavaScript functions.

Each file focuses on a single concept or problem to ensure clarity and better understanding.

---

## 🎯 Purpose

* Strengthen understanding of function concepts
* Practice real-world scenarios
* Improve problem-solving skills
* Build clean and structured coding habits

---

## 📂 Structure

Each file follows a simple pattern:

* Question description
* Code implementation
* Output (where applicable)

---

## 🧠 Topics Covered

* Function Basics
* Parameters & Arguments
* Default Parameters
* Rest Operator
* Return Values
* Closures
* Higher Order Functions
* Pure Functions
* IIFE
* Real-world Problems (BMI, Discount, etc.)

---

## 🚀 How to Use

* Open any file
* Read the question
* Understand the logic
* Try modifying the code

---

## 📌 Note

Each question is written in a separate file for better readability and easy revision.
