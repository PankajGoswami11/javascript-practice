# JavaScript Practice

This repository contains some basic JavaScript practice problems that I solved while learning the fundamentals of the language.  
The goal of this practice is to improve problem-solving skills and get more comfortable with core JavaScript concepts.

## Topics Covered

The following concepts are used in these exercises:

- Variables (`let`, `const`)
- Functions
- Arrow Functions
- Conditional Statements (`if/else`)
- Switch Case
- Ternary Operator
- Objects
- Basic Math Operations
- JavaScript Built-in Methods

## Practice Problems

1. Even or Odd (using modulus operator)
2. Sum of Two Numbers
3. Area of a Rectangle
4. Square of a Number
5. Average of Three Numbers
6. Temperature Converter (Celsius to Fahrenheit)
7. Remainder Calculator
8. Simple Calculator
9. Find the Largest Number
10. Power Calculator
