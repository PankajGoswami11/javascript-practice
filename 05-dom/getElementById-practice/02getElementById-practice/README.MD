## 📘 Questions List

### 🔹 Q26–Q30

* Get input value and display it in `#output`
* Clear input field
* Toggle visibility of `#box`
* Change page background color
* Count button clicks

### 🔹 Q31–Q35

* Add `<li>` dynamically
* Remove last `<li>`
* Update all `<p>` elements
* Disable button
* Enable button

### 🔹 Q36–Q40

* Toggle image source
* Show current date & time
* Character count
* Convert text to lowercase
* Hover effect on `#box`

### 🔹 Q41–Q45

* Remove all `<li>` from list
* Clone `#box`
* Add multiple classes
* Check class existence
* Scroll to top

### 🔹 Q46–Q50

* Change input placeholder
* Create button dynamically
* Count `<li>` elements
* Replace element
* Detect keyboard key press
