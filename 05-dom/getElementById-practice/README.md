## 📌 About This Project

This repository contains a collection of **17 structured JavaScript DOM practice problems** using `getElementById`.

It is designed to help developers build a strong foundation in DOM manipulation with clean and practical examples.

---

## 🎯 What You Will Learn

✨ Selecting elements using `getElementById`
✨ Changing text and HTML dynamically
✨ Updating styles using JavaScript
✨ Working with attributes (`src`, `href`)
✨ Handling user input
✨ Adding and removing classes
✨ Creating and appending elements
