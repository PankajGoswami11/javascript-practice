## 📌 About This Project

This repository is a **complete JavaScript DOM practice collection (Q1–Q50)** designed to build strong fundamentals in DOM manipulation.

It starts with **basic `getElementById` concepts (Q1–Q25)** and progresses to **event handling, dynamic elements, and real-world interactions (Q26–Q50)**.

---

## 🎯 What You Will Learn

✨ DOM selection using `getElementById`
✨ Text & HTML manipulation
✨ Styling elements dynamically
✨ Working with attributes (`src`, `href`, `value`)
✨ Handling user input and events
✨ Class manipulation (`add`, `remove`, `contains`)
✨ Creating, appending, and replacing elements
✨ Keyboard and mouse event handling
